Requirements

  1. jQuery
  2. Bootstrap 4
  3. template.js file
  4. Template files
  5. Structure as below

Page Structure

Every page is separated into several parts, each part is loaded in a main page at runtime to form a final page viewable to the end user.

These parts can be as below.

  1. The page itself which will dynamically combine all template and data files and will be called by the user
  2. Body template file
    • The template files are loaded in two stages, initially a base template is loaded which then loads all the rest of the template files
    • The body template file is the base template file
  3. head template file
    • Contains the contents of the <head>; usually the css files and html configurations
  4. nav template file
    • Contains the top navigation html, you can edit it to add/remove buttons and links
  5. footer template file
    • Contains the footer of the pages
  6. Page Content file
    • Each page should have its own content file and contains the actual data content of the page
  7. template.js file
    • It contains the jQuery code to combine all files and show a final page to user.

Directory Structure

All the above page parts are saved in several directories as below.

  1. css and js
    • Contains the css style files in css and javascript files in js directory
  2. data
    • Contains the files for data part of the pages
  3. pages
    • The main pages that are available to the user, these pages contain the javascript to load their content at runtime from data and template directories
  4. template
    • It contain all the template files
  5. index.html
    • The index page to load the website

How to use

The template can be used by editing files at minimum four places

  1. Navigation Menu
    • You have to edit the template/nav.html to update it to have your required links in the top navigation bar
  2. Footer
    • You have to edit the template/footer.html to update it to have a footer for your website
  3. main pages
    • The main pages are linked to the navigation links, when loaded they load the content from data directory and template components from template directory, You have to update the file names according to you requirements and then name of data page to be loaded in the javascript of these pages
  4. data pages
    • data pages are loaded by main pages in page directory as the page content. After editing the pages in pages directory you have to make/update data page in data directory, remember that the content of the page has to be included in a outer div component
