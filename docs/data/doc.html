Requirements

  1. jQuery 3.3.1
  2. Bootstrap 4.1.3
  3. Template file
  4. Content files
  5. Structure as below

Page Structure

Every page viewable to the user is separated into two parts, one is the master page and the other one is the content page.

At backend the master page is placed at one location and it is only one page and content pages are separately located at another location.

At runtime the master page is loaded initially and will load the home content page at default and will make links to load ans show the other pages using page links.

Directory Structure

All the above page parts are saved in several directories as below.

  1. css
    • Contains the css style files in css and javascript files in js directory
  2. data
    • Contains the content files each file contains content part of a separate page
  3. pages
    • The folder to contain the master page
  4. index.html
    • The index page to load the website

How to use

The template can be used by editing files at minimum two places

  1. master page
    • This page is located in pages folder, you can edit this page to update the overall website look, menus etc
  2. content/data pages
    • content pages are located in data folder. These pages are loaded by main pages in page directory as the page content. After editing the pages in pages directory you have to make/update data pages in data directory, remember that the content of each page has to be included in a outer div component
